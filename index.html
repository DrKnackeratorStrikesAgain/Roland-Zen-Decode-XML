Roland Zen Decode XML

Decodes Roland editor XML (ZenCore, Jupiter-X/Xm, Juno-X, Verselab MV-1, etc.) into reference tables and structured data: byte layout, SYSEX offsets, and parameter metadata.

Source code & generator: github.com/DrKnackeratorStrikesAgain/Roland-Zen-Decode-XML. See the README for more information.

JSON / JS / Python / C header exports are in the repository under out_* directories; they are not listed in this folder but are produced by the same pipeline documented in the README.
